## core LDAP

Módulo web que gestiona un LDAP. Dibuja graficamente la estructura del LDAP.
Por otro lado expone los formularios para la gestión de cada objeto del LDAP.

Debe estar desacoplado de la capa de servicio que es la que acceda al LDAP mediante el módulo core-ldap.
